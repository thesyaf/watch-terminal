## Overview
This is a single-page web application designed to simulate a retro terminal experience. It's intended to be served from an NFC chip embedded in a Casio watch. When scanned, it runs an initial "hacking" animation, then presents an interactive command prompt.

## Features
- Retro CRT terminal aesthetic
- Typing animation for initial sequence
- Interactive command line interface
- Basic commands: `help`, `clear`, `date`
- Designed for mobile viewing

## Access the Live App
The live application is hosted on GitHub Pages and can be accessed at:
`https://thesyaf.github.io/watch-terminal/`

## How to Use
1.  **Scan the Watch:** (Or navigate to the URL above)
2.  **Watch the Intro:** The "hacking" animation will play.
3.  **Start Typing:** After the animation, a `$` prompt will appear. Click anywhere on the terminal to ensure the input is focused, then type commands.
4.  **Available Commands:**
    *   `help`: Displays available commands.
    *   `clear`: Clears the terminal screen.
    *   `date`: Shows the current date and time.
